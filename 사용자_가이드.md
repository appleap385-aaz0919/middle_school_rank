# 지역별 중학교 순위 서비스 사용자 가이드

## 서비스 개요

본 서비스는 학부모와 학생을 위한 지역별 중학교 순위 조회 서비스로, 국가수준 학업성취도 평가 데이터를 기반으로 학교 순위를 제공합니다.

## 시작하기

### 1. 서비스 접속

서비스에 접속하면 로그인 페이지가 표시됩니다.
- 로컬 개발 환경: http://localhost:5174

### 2. 회원가입

**최초 사용자는 회원가입이 필요합니다.**

1. 로그인 페이지에서 "회원가입" 링크 클릭
2. 필수 정보 입력:
   - **이메일**: 사용하실 이메일 주소 입력
   - **비밀번호**: 최소 6자 이상 입력
   - **학교급**: 초등/중등/고등 중 선택
   - **학년**: 학교급에 따라 학년 선택 (초등: 1~6, 중등: 1~3, 고등: 1~3)
3. 모든 필드를 올바르게 입력하면 "회원가입" 버튼이 활성화됩니다.
4. 회원가입 버튼 클릭 시 가입 완료 및 로그인 페이지로 자동 이동

### 3. 로그인

1. 이메일과 비밀번호 입력
2. "로그인" 버튼 클릭
3. 인증 성공 시 메인 페이지로 이동

## 주요 기능 사용법

### 메인 페이지

로그인 후 메인 페이지에서 다음 기능을 이용할 수 있습니다.

#### 1. 연도 선택

- 우상단의 연도 선택 드롭다운에서 2023년, 2024년, 2025년 중 선택
- 기본값은 2025년으로 설정되어 있습니다.
- 연도를 변경하면 해당 연도의 데이터로 조회됩니다.

#### 2. 지역 선택 (두 가지 방식 제공)

**방식 1: 단계별 선택**

1. "단계별 선택" 탭 클릭
2. 첫 번째 드롭다운에서 시/도 선택 (예: 서울특별시)
3. 두 번째 드롭다운에서 지역구 선택 (예: 강남구)
4. "검색" 버튼 클릭

**방식 2: 직접 검색**

1. "직접 검색" 탭 클릭
2. 검색창에 지역구명 입력 (최소 2글자 이상)
   - 예: "강남", "서초", "성남시"
3. "검색" 버튼 클릭
4. 검색 결과에서 원하는 지역을 클릭하여 선택

### 3. 학교 순위 조회

지역을 선택하면 해당 지역의 중학교 순위가 테이블 형태로 표시됩니다.

**테이블 정보:**
- **순위**: 전국 기준 순위 (1~3위는 금, 은, 동 뱃지로 표시)
- **학교 이름**: 정식 학교명
- **설립 구분**: 공립/사립
- **남녀 공학**: 남학교/여학교/남녀공학
- **학업성취도 평균**: '보통학력' 이상 비율(%)
- **특목고/자사고 진학률**: 진학률(%)

**조회 시점**: 현재 날짜와 시간이 상단에 표시됩니다.

### 4. 로그아웃

- 모든 페이지 상단 우측에 "로그아웃" 버튼이 표시됩니다.
- 로그아웃 버튼 클릭 시 세션이 종료되고 로그인 페이지로 이동합니다.

## 데이터 설명

### 순위 산정 기준

- **데이터 출처**: 학교알리미 공시 데이터
- **순위 기준**: 국가수준학업성취도평가 '보통학력' 이상 비율
- **연도별 데이터**: 각 연도별로 해당 연도의 데이터를 사용

### 제공되는 지역

현재 다음 지역의 데이터를 제공합니다:

- 서울특별시: 강남구, 서초구, 송파구 등
- 경기도: 성남시, 수원시 등
- 부산광역시: 해운대구 등
- 대구광역시: 수성구 등
- 인천광역시: 남동구 등
- 광주광역시: 북구 등
- 대전광역시: 유성구 등

※ 샘플 데이터로 구성되어 있어 모든 지역의 데이터가 포함되어 있지 않습니다.

## 주의사항

1. **이메일 중복**: 동일한 이메일로 중복 가입이 불가능합니다.
2. **비밀번호**: 최소 6자 이상이어야 합니다.
3. **검색**: 직접 검색 시 최소 2글자 이상 입력해야 합니다.
4. **데이터**: 현재 샘플 데이터로 제공되며, 실제 학교알리미 API 연동 시 더 많은 데이터를 제공할 수 있습니다.

## 기술 스택

- **프레임워크**: React + Vite
- **인증**: Firebase Authentication
- **라우팅**: React Router
- **스타일**: CSS (반응형 디자인)
- **데이터**: 샘플 데이터 (JSON 형식)

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 문제 해결

### 로그인이 되지 않을 때

1. 이메일과 비밀번호가 올바르게 입력되었는지 확인
2. 이미 회원가입이 되어 있는지 확인
3. 인터넷 연결 상태 확인 (Firebase 인증 필요)

### 검색 결과가 없을 때

1. 지역명이 올바르게 입력되었는지 확인
2. 해당 지역의 데이터가 존재하는지 확인 (샘플 데이터 제한)

### 페이지가 로딩되지 않을 때

1. 개발 서버가 실행 중인지 확인
2. 브라우저 콘솔에서 에러 메시지 확인
